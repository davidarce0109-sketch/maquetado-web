git init
(sirve para iniciar el repositorio local)

git config --local user.name "mi nombre"
(esto configura el nombre de usuario localmente)

git config --local user.email "mi email de github"
(esto configura el email de usuario asociado a la cuenta de github)

git branch -M main
(cambiar el nombre de la ramma master a main)

git add .
(añade todos los archivos al reposiorio)


git commit -m "version de subida"
(es la version que usted  esta subiendo al repositorio)

git remote add origin 
despues de origin das espacio y escribe la direccion del repositorio a subir
(selecciona el repositorio al cual se subire el documento)

git push -u origin main
(este empuja los cambios locales a la nube)

git status
(comprueba el estado de la rama)

git clone
espacio escribe URL del repositotio
(descarga en una carpta lo que hay en el repositorio)
